## Time series forecasting

 

In this project, we present a model that helps to forecast stock exchange by using facebook's prophet package. 

## Features

* 'Date'
* 'SecuritiesCode'
* 'Open'
* 'High'
* 'Low'
* 'Close'
* 'Volume'
* 'AdjustmentFactor'
* 'SupervisionFlag'
